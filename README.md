# Part 3 — Python File I/O and HTTP Error Handling

A Python project that covers file operations, API integration, exception handling, and logging.

## Project Structure

python-assignment-part3/
├── part3_api_files.py
├── python_notes.txt
└── error_log.txt

## Tasks Covered

### Task 1 — File Read & Write
- Writes and appends notes to python_notes.txt
- Reads file back with line numbers
- Keyword search functionality

### Task 2 — API Integration
- Fetches products from DummyJSON API
- Filters and sorts by rating and price
- Searches by category
- POST request simulation

### Task 3 — Exception Handling
- Safe divide function
- Guarded file reader with finally block
- Robust API calls with error handling
- Input validation loop

### Task 4 — Logging
- Timestamped error logging to error_log.txt
- Logs ConnectionError and HTTP 404 errors

## How to Run

pip install requests
python part3_api_files.py
